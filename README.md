# Twon

## Requirements

- Firefox: 52+
- Chrome: 52+
- Edge: 14+
- Opera: 39+
- Safari: 10.1+
- Android Browser: 109+
- Samsung Internet: 6.2+
- Firefox for Android: 110+
- iOS Safari: 10.3+
- Opera Mobile: 73+
- ~~Internet Explorer~~
